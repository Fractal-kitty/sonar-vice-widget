## 📋 What is sonar-vice-widget?

sonar-vice-widget is a small desktop tool designed for SteelSeries GG users. It stays on top of other windows so you can easily adjust Sonar audio settings, switch EQ profiles, and control RGB lighting. The widget floats on your screen and uses simple controls to manage these features without opening the full SteelSeries software.

The main features include:  
- Quick audio control for Sonar  
- Easy switching between EQ profiles  
- Simple RGB lighting adjustments  
- Always-on-top window for fast access  

This app is built for Windows and works with SteelSeries devices using the Sonar and GameSense APIs.

## 🖥️ System Requirements

To run sonar-vice-widget on your Windows computer, make sure your system meets these conditions:

- SteelSeries device compatible with Sonar and GameSense

## ⚙️ How to Use sonar-vice-widget

The widget interface has simple buttons and sliders.

### Audio Controls

- Volume slider: Adjust your Sonar audio volume smoothly.  
- Mute button: Quickly mute or unmute your device.  
- Microphone toggle: Turn your mic on or off with a single click.

### EQ Profiles

- Drop-down menu: Select your preferred equalizer profile.  
- Add New Profile: Open SteelSeries GG to create or edit profiles.  

### RGB Lighting

- Color picker: Choose colors for your SteelSeries device RGB lighting.  
- Modes: Switch between lighting effects like static, breathing, or reactive.

Each change happens instantly and updates your device through SteelSeries software.

## 🛠️ Troubleshooting

If sonar-vice-widget does not work as expected, try these tips:

- Make sure your SteelSeries GG software is up to date.  
- Restart your computer to ensure all drivers are loaded correctly.  
- Check that your SteelSeries device is properly connected via USB.  
- If the widget does not appear, open Task Manager and make sure `sonar-vice-widget.exe` is running.  
- Disable any firewall or antivirus software temporarily if the app fails to communicate with SteelSeries hardware.  
- Re-install the app if problems persist.
